About LIBOT

LIBOT is an application developed by CK3, a mapping intelligence that uses the languages of Python and JavaScript to implement a seamless way to explore maps using different types of street view according to the preference of our users.


The application also includes real-time tracking, a search function to find any location in the map, and a measuring distance logic to understand how far the destination is. LIBOT ensures that users should always know where they are at any place, at any time.
